Euphoria

MERN stack

For my final GA project, I created an app that retrieves concert data from the skiddle API and allows users to track all of the concerts they want to go to in London. They can filter and sort the gigs by parameters such as name and date. I also used the google Maps API, to display the precise location of any given concert.

ICON

React, Redux, Firebase, Node.js

Icon is a web app for folks in the energy business. Subscribers can view insights uploaded by the admin, participate in a forum, schedule events in a calendar or upload a legal register to check their compliance with the law. I joined this project a year ago, helping to maintain the current codebase and build new features.

thingdex

Expo, Firebase, Node.js

thingdex is an app I have been working on that allows users to save and recommend links on the internet. You can enter (almost) any link, and its metadata (such as image and title) will be parsed and stored in one place so you can have a catalogue of all of your favourite things.

Node.js MongoDB, Mongoose, Express.js, EJS, Bootstrap 4, SCSS, JavaScript
