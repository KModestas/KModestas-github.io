Euphoria

MERN stack

I created an app that retrieves concert data from the skiddle API and allows users to track all of the concerts they want to go to in London. They can filter and sort the gigs by parameters such as name and date. I also used the google Maps API, to display the precise location of any given concert.

SPLTN

MEAN stack

This app involved working directly with two of my classmates. We built a Full Stack Application that allows groups of individuals to log a trip and split the accumulated bills on that trip to determine how much money is owed to each individual and the total spend of the holiday. We used the Google Maps and Chart.js API to accomplish this goal.

Superfoods

Node.js MongoDB, Mongoose, Express.js, EJS, Bootstrap 4, SCSS, JavaScript

This was my first Full-stack application which involved building a fully RESTful application containing all CRUD functionalities. I built an app displaying an index of foods with their macronutrients which users can upload or find after registering and logging in.

Chimneys & Candy Canes

HTML5, CSS3, JavaScript (ES6), jQuery

This was my first Front End project which involved building a JavaScript game. Since it was the holiday season, I decided to make a festive version of snakes and ladders which utilised all the fundamental concepts of JavaScript such as (but not limited to) functions, array, loops and objects.

Todo App

HTML5, CSS3, JavaScript (ES6), React, Redux

This was a simple Front End React / Redux app that I built, which allows users to log a list of items; check them off and reset the entire list. It utilises many of the fundamental concepts of Redux such as dispatching actions to reducers and updating the global state.

SuperSquad

React, Redux

I built this app primarily to further my React knowledge and to understand the core concepts of redux. It allows you to pick from a list of heroes which are displayed on the screen and then add them to be displayed on your own squad list.
